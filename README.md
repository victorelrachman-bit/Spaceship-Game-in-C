# Spaceship-Game-in-C

## Objetivo
  Jogo desenvolvido em C em que o jogador controla uma nave, desviando de meteoros que se tornam cada vez mais difíceis de desviar.

## Conceitos usados
  - Loop de jogo (game loop)
  - Gerenciamento de entidades (jogador + obstáculos)
  - Reinicialização de estado
  - Separação entre lógica e execução principal
  - Programação procedural aplicada a jogos

## Orientações
  - Compile com gcc ou DevC++, com o arquivo main.cpp na mesma pasta do arquivo mov.h, ou simplesmente rode o arquivo .exe
  - Por mais que o arquivo seja c++, todo projeto foi desenvolvido conforme a linguagem C, sem utilizar recursos específicos do C++

## Como jogar
  - Use as teclas w para subir e s para descer a nave
